# 🖥️ DELITECH IT CLUB – Registration System

> A **real-time, zero-cost** event registration system built for 300+ students. Features photo uploads, live analytics dashboard, role-based access, and CSV export – all powered by Next.js, Supabase, and Vercel.

## 🚀 Live Demo
🔗 **[Live Registration](https://club-registration-system-indol.vercel.app/register)**  
🔐 **[Dashboard](https://club-registration-system-indol.vercel.app/dashboard)** (Contact admin for access)

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📝 **Student Registration** | Mobile-friendly form with photo upload and file validation (< 1 MB) |
| 🖼️ **Photo Storage** | Uploaded selfies stored securely in Supabase Storage |
| 📊 **Analytics Dashboard** | Real-time charts, leaderboards, and student gallery |
| 🔐 **Role-Based Access** | Admin, President, VP, and Lecturer roles with unique passwords |
| 🪪 **Glassmorphism UI** | Modern, immersive interface with 3D spinning logo |
| 📥 **CSV Export** | One-click export of student data for reporting |
| 🔄 **Real-Time Sync** | Dashboard auto-updates when a new student registers |
| 📱 **Mobile Responsive** | Works flawlessly on phones, tablets, and desktops |
